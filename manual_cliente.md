# Manual de Usuario - Cliente

Sistema: Libreria Bartleby  
Version: 29/05/2026  
Dirigido a: Clientes registrados y visitantes del catalogo

## 1. Proposito del manual

Este manual explica como una persona cliente puede usar el sistema de Libreria Bartleby para explorar el catalogo, crear una cuenta, iniciar sesion, apartar libros, consultar sus reservas, guardar su ticket y administrar su perfil.

El manual esta escrito para usuarios finales. No requiere conocimientos tecnicos.

## 2. Alcance del usuario cliente

El cliente puede realizar las siguientes acciones:

- Consultar la informacion publica de la libreria.
- Buscar libros en el catalogo.
- Filtrar libros por genero y precio.
- Revisar la ficha completa de un ejemplar.
- Crear una cuenta.
- Activar su cuenta mediante codigo de verificacion.
- Iniciar sesion con contrasena.
- Iniciar sesion con codigo enviado por correo.
- Recuperar su contrasena.
- Agregar libros disponibles a la bolsa.
- Modificar la cantidad de ejemplares en la bolsa.
- Quitar libros de la bolsa.
- Confirmar una reserva.
- Consultar sus reservas.
- Abrir el ticket de una reserva.
- Guardar el ticket como imagen.
- Cancelar reservas pendientes, cuando la opcion este disponible.
- Editar datos de su perfil.
- Consultar historial de reservas y compras asociadas a su cuenta.
- Consultar el aviso de privacidad y los terminos y condiciones.

El cliente no puede administrar inventario, proveedores, ventas, reportes, personal ni auditoria.

## 3. Requisitos para usar el sistema

Para usar el sistema como cliente se necesita:

- Un navegador web actualizado.
- Conexion a internet.
- Una cuenta registrada en Libreria Bartleby para poder apartar libros.
- Un correo electronico valido para recibir codigos de verificacion.

El catalogo puede consultarse sin iniciar sesion, pero para apartar libros es necesario tener una cuenta activa.

## 4. Pantalla principal y navegacion

Al entrar al sistema, el cliente encuentra una barra superior con accesos principales.

Opciones habituales:

- **Libreria Bartleby**: lleva al inicio/catalogo.

- **Catalogo**: abre el listado de libros disponibles.
- **Bolsa**: muestra los libros seleccionados para apartado.
- **Mis reservas**: muestra las reservas del cliente autenticado.
- **Mi perfil**: muestra datos personales, reservas e historial.
- **Ingresar**: aparece cuando el usuario no ha iniciado sesion.
- **Cerrar sesion**: aparece cuando el usuario ya inicio sesion.
- **Modo claro/oscuro**: permite cambiar el tema visual.

## 5. Crear una cuenta de cliente

El registro permite crear una cuenta para apartar libros y consultar historial.

### 5.1 Entrar al formulario de registro

1. Desde la barra superior, seleccionar **Ingresar**.
2. En la pantalla de acceso, seleccionar la opcion **Registrate** o equivalente.
3. El sistema abre la pantalla **Crear Cuenta**.

[Captura sugerida: pantalla de login con enlace a registro.]

### 5.2 Capturar datos personales

En el formulario de registro se solicitan:

- Nombre.
- Apellido.
- Usuario.
- Telefono.
- Correo electronico.
- Direccion o referencias, opcional.
- Contrasena.
- Confirmacion de contrasena.

Pasos:

1. Escribir nombre y apellido.
2. Definir un nombre de usuario.
3. Capturar telefono.
4. Capturar un correo electronico valido.
5. Capturar direccion o referencias, si aplica.
6. Escribir una contrasena segura.
7. Repetir la contrasena.

[Captura sugerida: formulario Crear Cuenta completo antes de marcar aviso de privacidad.]

### 5.3 Aceptar aviso de privacidad y terminos

Antes de crear la cuenta, el cliente debe marcar la casilla donde acepta:

- **Aviso de Privacidad**.
- **Terminos y Condiciones**.

El boton **Crear cuenta** permanece deshabilitado hasta que se marque esta casilla.

Pasos:

1. Leer los documentos enlazados si es necesario.
2. Marcar la casilla de aceptacion.
3. Verificar que el boton **Crear cuenta** se active.
4. Presionar **Crear cuenta**.

[Captura sugerida: casilla de aceptacion marcada y boton Crear cuenta activo.]

### 5.4 Activar cuenta con codigo

Despues del registro, el sistema genera un codigo de verificacion y lo envia al correo del cliente.

Pasos:

1. Revisar la bandeja de entrada del correo registrado.
2. Copiar el codigo de 6 digitos recibido.
3. Volver a la pantalla de verificacion.
4. Escribir el codigo.
5. Confirmar.

Si el codigo es correcto, la cuenta queda activa y el sistema permite iniciar sesion.

Si el codigo no llega, el cliente puede solicitar reenvio desde la misma pantalla, si la opcion esta disponible.

[Captura sugerida: pantalla de verificacion de codigo despues del registro.]

## 6. Iniciar sesion

El sistema permite iniciar sesion de dos formas: con usuario y contrasena, o con codigo enviado por correo.

### 6.1 Inicio de sesion con contrasena

Pasos:

1. Seleccionar **Ingresar**.
2. Capturar usuario y contrasena.
3. Presionar el boton de ingreso.

Si los datos son correctos, el sistema abre el catalogo o el area principal disponible para el usuario.

[Captura sugerida: formulario de inicio de sesion con usuario y contrasena.]

### 6.2 Inicio de sesion con codigo por correo

Esta opcion permite entrar sin escribir contrasena, usando un codigo temporal enviado al correo.

Pasos:

1. En la pantalla de acceso, seleccionar la opcion de ingresar con codigo.
2. Escribir el correo electronico registrado.
3. Presionar el boton para solicitar codigo.
4. Revisar el correo.
5. Capturar el codigo recibido.
6. Confirmar.

Si el codigo coincide, el sistema inicia sesion automaticamente.

[Captura sugerida: pantalla Login con codigo, solicitando correo electronico.]

[Captura sugerida: pantalla para verificar codigo de login.]

## 7. Recuperar contrasena

Si el cliente olvida su contrasena, puede solicitar recuperacion por correo.

Pasos:

1. Entrar a **Ingresar**.
2. Seleccionar **Recuperar contrasena**.
3. Escribir el correo electronico registrado.
4. Enviar la solicitud.
5. Revisar el codigo recibido por correo.
6. Capturar el codigo en la pantalla de recuperacion.
7. Escribir la nueva contrasena.
8. Confirmar la nueva contrasena.
9. Guardar el cambio.

Despues de actualizar la contrasena, el cliente puede iniciar sesion normalmente.

## 8. Consultar el catalogo

El catalogo muestra los libros disponibles para venta o apartado.

### 8.1 Entrar al catalogo

Pasos:

1. Desde la barra superior, seleccionar **Catalogo**.
2. El sistema muestra la pantalla **Descubre tu proxima gran lectura**.
3. Debajo del encabezado aparece el buscador con filtros.

[Captura sugerida: pantalla completa del catalogo con encabezado, buscador y primeras tarjetas de libros.]

### 8.2 Buscar por titulo, autor o ISBN

Pasos:

1. Ubicar el campo **Buscar libro, autor o ISBN**.
2. Escribir una palabra clave, por ejemplo:
   - Nombre del autor.
   - Titulo del libro.
   - ISBN.
   - Palabra incluida en la descripcion.
3. Presionar **Buscar**.

El sistema muestra solo los ejemplares que coinciden con la busqueda.

[Captura sugerida: busqueda por autor o titulo con resultados filtrados.]

### 8.3 Filtrar por genero

Pasos:

1. Abrir el selector **Genero**.
2. Elegir una categoria.
3. Presionar **Buscar**.

El sistema muestra los libros que pertenecen al genero seleccionado.

[Captura sugerida: selector de Genero desplegado.]

### 8.4 Filtrar por precio

El cliente puede filtrar por precio minimo y precio maximo.

Pasos:

1. Escribir un valor en **Precio Min**, si desea buscar desde cierta cantidad.
2. Escribir un valor en **Precio Max**, si desea limitar el precio.
3. Presionar **Buscar**.

Ejemplo:

- Precio Min: `100`
- Precio Max: `300`

El sistema mostrara libros con precio dentro de ese rango.

[Captura sugerida: filtros de precio llenos y resultados visibles.]

### 8.5 Limpiar filtros

Cuando hay filtros activos, el sistema muestra un boton con icono de cerrar o limpiar.

Pasos:

1. Presionar el boton de limpiar filtros.
2. El catalogo vuelve a mostrar todos los libros disponibles.

## 10. Usar la bolsa

La bolsa contiene los libros seleccionados antes de confirmar la reserva.

Importante: agregar un libro a la bolsa todavia no confirma el apartado. El apartado se genera hasta presionar **Confirmar Reserva**.

### 10.1 Abrir la bolsa

Pasos:

1. Presionar **Bolsa** en la barra superior.
2. Revisar la pantalla **Tu Seleccion de Lecturas**.

[Captura sugerida: pantalla Bolsa con uno o varios libros agregados.]

### 10.2 Revisar los libros seleccionados

Cada articulo en la bolsa muestra:

- Portada.
- Titulo.
- Autor.
- Estado fisico.
- Precio unitario.
- Cantidad.
- Subtotal.
- Opcion para agregar mas unidades.
- Opcion para quitar el articulo.

[Captura sugerida: tarjeta de un libro dentro de la bolsa, mostrando cantidad y subtotal.]

Pasos:

1. Ubicar el libro en la bolsa.
2. En el campo **Cantidad**, escribir cuantas unidades adicionales se quieren agregar.
3. Presionar el boton `+`.

El sistema suma esa cantidad a la cantidad ya existente, siempre que haya stock disponible.

Ejemplo:

- Si la bolsa ya tiene 1 unidad.
- El cliente escribe 2 y presiona `+`.
- La bolsa intenta quedar con 3 unidades.

Si no hay stock suficiente, el sistema muestra una advertencia.

## 11. Confirmar una reserva

La reserva se crea a partir del contenido actual de la bolsa.

### 11.1 Antes de confirmar

El cliente debe revisar:

- Que los libros sean correctos.
- Que las cantidades sean correctas.
- Que el total sea correcto.
- Que pueda recoger los libros dentro del periodo establecido.

La reserva tiene vigencia de 72 horas desde su confirmacion.

### 11.2 Confirmar reserva

Pasos:

1. Entrar a **Bolsa**.
2. Revisar los libros y el total.
3. Presionar **Confirmar Reserva**.
4. Esperar el mensaje de confirmacion.

El sistema:

- Verifica disponibilidad de stock.
- Crea la reserva.
- Genera un codigo de reserva tipo `BART-XXXX`.
- Limpia la bolsa.
- Redirige al cliente a **Mis reservas**.

[Captura sugerida: boton Confirmar Reserva antes de presionarlo.]

[Captura sugerida: mensaje de reserva confirmada.]

### 11.3 Cuando no hay stock suficiente

Puede ocurrir que otro usuario haya reservado el mismo ejemplar antes.

Si el stock ya no alcanza, el sistema muestra un mensaje de error y no crea la reserva.

Acciones recomendadas:

1. Volver a la bolsa.
2. Quitar el ejemplar no disponible.
3. Buscar otra opcion en el catalogo.
4. Confirmar de nuevo.

[Captura sugerida: mensaje de error por stock insuficiente, si se presenta.]

## 12. Politica de apartados para cliente

### 12.1 Vigencia

Despues de confirmar, los libros quedan reservados por 72 horas.

Durante ese tiempo:

- El libro queda apartado para el cliente.
- Otros clientes no pueden reservar ese mismo ejemplar si no queda stock libre.
- El cliente debe acudir a la libreria para pagar y recoger.

### 12.2 Pago

El sistema indica **pago contra entrega**.

Esto significa:

- No se cobra en linea.
- El pago se realiza directamente en sucursal.
- El cliente debe presentar su codigo o ticket de reserva.

### 12.3 Vencimiento

Si el cliente no recoge la reserva dentro del periodo establecido:

- La reserva puede ser cancelada.
- Los libros vuelven al catalogo.
- Otro cliente puede apartarlos.

### 12.4 Cancelacion

El cliente puede cancelar reservas pendientes cuando la opcion este disponible.

No se pueden cancelar reservas que ya esten completadas o canceladas.

## 15. Cancelar una reserva pendiente

La cancelacion permite liberar libros que el cliente ya no desea recoger.

### 15.1 Condiciones

Solo se pueden cancelar reservas en estado **Pendiente**.

No se pueden cancelar:

- Reservas completadas.
- Reservas ya canceladas.
- Reservas que el personal ya proceso como venta.

### 15.2 Pasos para cancelar

1. Entrar a **Mis reservas**.
2. Ubicar la reserva pendiente.
3. Seleccionar la opcion **Cancelar**, si aparece disponible.
4. Confirmar la accion.

El sistema cambia la reserva a cancelada y devuelve los ejemplares al catalogo.

[Captura sugerida: mensaje de reserva cancelada correctamente.]

## 20. Preguntas frecuentes

### 20.1 Puedo apartar sin cuenta?

No. El catalogo puede consultarse sin cuenta, pero para apartar libros es necesario registrarse e iniciar sesion.

### 20.2 El pago se hace en linea?

No. El sistema maneja pago contra entrega. El cliente paga en sucursal al recoger los libros.

### 20.3 Cuanto dura una reserva?

La reserva dura 72 horas desde que se confirma.

### 20.4 Que pasa si no recojo mi reserva?

La reserva puede cancelarse y los libros vuelven a estar disponibles en el catalogo.

### 20.5 Puedo cancelar una reserva?

Si la reserva esta pendiente, puede cancelarse cuando la interfaz muestre la opcion. Si no aparece, se debe contactar al personal.

### 20.6 Por que un libro aparece como reservado?

Porque otro cliente lo aparto previamente. Si el contador llega a cero o el personal libera la reserva, el libro puede volver a estar disponible.

### 20.7 Que hago si no llega el codigo de verificacion?

Revisar spam o correo no deseado. Si la pantalla permite reenviar el codigo, usar esa opcion. Si el problema continua, contactar al personal.

### 20.8 Por que no puedo agregar mas unidades?

Puede deberse a falta de stock disponible o al limite maximo de ejemplares por reserva.

### 20.9 Que debo presentar en tienda?

El codigo de reserva, el ticket digital o la imagen descargada del ticket.

### 20.10 Puedo cambiar mis datos?

Si. Desde **Mi perfil** se pueden actualizar datos personales cuando el formulario esta disponible.

## 21. Errores comunes y solucion

### 21.1 El sistema me pide iniciar sesion al anadir un libro

Causa: no has iniciado sesion.  
Solucion: inicia sesion o crea una cuenta.

### 21.2 El codigo de verificacion es incorrecto

Causa posible: se escribio mal, expiro o se genero uno nuevo.  
Solucion: revisar el ultimo correo recibido y volver a intentarlo.

### 21.3 La bolsa esta vacia despues de confirmar

Causa: esto es normal.  
Solucion: al confirmar una reserva, la bolsa se limpia y la reserva pasa a **Mis reservas**.

### 21.4 El libro ya no esta disponible

Causa: otro cliente lo reservo o el stock cambio.  
Solucion: quitarlo de la bolsa y elegir otro ejemplar.

### 21.5 No puedo ver el ticket

Causa posible: no has iniciado sesion o intentas abrir una reserva de otro usuario.  
Solucion: inicia sesion con la cuenta correcta.

## 24. Glosario

**Apartado**: reserva temporal de uno o varios libros para recoger y pagar despues en sucursal.

**Bolsa**: seleccion temporal de libros antes de confirmar una reserva.

**Catalogo**: listado de libros disponibles o visibles para el cliente.

**Codigo de reserva**: identificador unico de la reserva, con formato similar a `BART-XXXX`.

**Ejemplar**: copia especifica de un libro. Un mismo titulo puede tener varios ejemplares.

**Estado fisico**: condicion del ejemplar, por ejemplo nuevo, como nuevo o aceptable.

**SKU**: codigo interno del ejemplar.

**Ticket**: comprobante digital de la reserva.

**Stock disponible**: cantidad real que puede apartarse, descontando reservas activas.
